# CounterSignal

Offensive security payload generator for testing AI system defenses. Indirect prompt injection, context file poisoning, and RAG retrieval poisoning.

## Tools

- **IPI** — indirect prompt injection payload generator (34 techniques, 7 formats, 7 payload types)
- **CXP** — context file poisoning for coding assistants
- **RXP** — RAG retrieval poisoning optimizer

## Quick start

```bash
pip install countersignal
countersignal ipi generate --technique all --format markdown
```
